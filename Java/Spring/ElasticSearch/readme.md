# 엘라스틱 서치
## 안내
기존 DB에서 검색을 진행할 때, 데이터가 많거나 인덱스 등을 사용하지 않으면 풀스캔으로 시간이 많이 걸린 경험이 있음<br/>
또한, 인덱스를 사용해서 검색 범위를 잘 조절하지 않으면 풀스캔을 사용해서 검색하기에 이를 개선할 방법이 필요해서 찾던 중 엘라스틱 서치에 대해 알게 되었음<br/>
이번 실습은 엘라스틱서치를 사용해서 기존 검색 방식을 개선하는 것이 목적임

## 개발 환경
1. JDK 21
2. Spring Boot 3
3. MariaDB
